fun Modifier.zoom(zoomable: ZoomableState, userSetupContentSize: Boolean = false, onLongPress: (Offset) -> Unit? = null, onTap: (Offset) -> Unit? = null): Modifier

A Modifier that can recognize gestures such as click, long press, double-click, one-finger zoom, two-finger zoom, drag, fling, etc., and then apply the gesture changes to the component. It can be used on any composable component.

Since it consumes all gestures, Modifier.clickable and Modifier.combinedClickable will not work. You can pass onTap and onLongPress parameters instead.

If the zoomed content does not fill the container, you can set the size, scaling and alignment of the content through the contentSize, contentScale, and alignment properties of ZoomableState. This will only translate within the content area after scaling.
